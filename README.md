# OmniMind

**Real-time Accessibility Translation for Physical & Cognitive Barriers**

A professional, modern web platform that helps people with disabilities interact with information more easily using AI.

## Features
- Beautiful, accessible UI
- Real-time chat demo with accessibility-focused AI
- Voice input/output support (Web Speech API)
- Dark / Light mode
- Fully responsive design
- Ready for API integration (OpenAI, Grok, Claude, etc.)

## Getting Started

1. Clone the repo
2. Open `index.html` in your browser
3. Try the live demo!
